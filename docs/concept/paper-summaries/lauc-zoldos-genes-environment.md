# Protein Glycosylation - An Evolutionary Crossroad Between Genes and Environment

Citation key: `lauc_zoldos_genes_environment`

Authors: Lauc; Zoldos

Year: 2010

Journal: Molecular BioSystems

Link: https://doi.org/10.1039/C0MB00067A

## Summary

This conceptual review frames protein glycosylation as a biological interface between genetic information and environmental responsiveness. Because glycans are produced by networks of enzymes and regulatory processes rather than by direct templates, glycan structures can change through gene expression, enzyme activity, localization, and environmental inputs.

The paper positions glycosylation as a flexible layer that can generate phenotypic variation without requiring direct changes in protein sequence.

## Relevance To This Project

This is a key conceptual predecessor for the evolved-interface part of the thesis. It supports the idea that glycan-output variation can mediate organism-environment interactions while leaving core genetic machinery conserved.

For the new paper, it helps justify shifting the expected evolvability signal toward regulatory and glycan-output layers.

## Use In Manuscript

Use in introduction and conceptual framing.

Use to distinguish the proposed architecture model from a narrow coding-sequence adaptation model.

## Limitations

The paper is broad and conceptual. It does not test N-glycosylation pathway regions, constraint gradients, disease architecture, or modern glycome GWAS evidence.
